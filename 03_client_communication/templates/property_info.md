# Template: Property Info

## When to Use
Sending a specific property or shortlist of properties to a buyer. Used during `property_search` stage when the agent wants to share options — either a single researched property or a curated set. May be the second or third touchpoint in a buyer relationship.

## Context to Read
- Property report(s) from `properties/[address_slug]/property_report.md` — pull key details for each property being shared
- `budget_range` — confirm each property is within range; note if any are at the ceiling
- `location_preferences` — confirm each property matches stated areas
- `constraints.tags` — any constraint that affects which properties make sense (e.g., `must_sell_first` means flexible close date matters; `lease_expiring` means move-in timeline matters)
- `timeline` — note if urgency affects which properties to prioritize

## Subject Line
- Single property: `[Address] — Worth a Look`
- Shortlist: `[X] Properties for You to Review`

## Required Content (in order)

1. **Brief framing** — one sentence explaining what you are sending and why. "Pulled three properties that fit your criteria" or "This one just came up and I think it's worth a look." Do not over-explain.

2. **For each property:**
   - Address and list price
   - Key specs (beds/baths/sqft, year built)
   - 1–2 sentences from the property report — what stands out, any flag worth knowing
   - Days on market if it affects urgency
   - One sentence on fit: how it measures against their budget and location preference

3. **Recommended next step** — which property to prioritize for a showing and why (if sending a shortlist). Or, if sending a single property, ask directly if they want to schedule a showing.

4. **Low-pressure close** — invite response without creating false urgency. Only add urgency if the property is genuinely at risk (very low DOM in a hot area, pending offer, etc.).

## Tone Notes
- Be honest about fit. If a property is at the top of their budget or has a flag, say so — do not hide it in hope they overlook it.
- One recommendation is better than presenting everything as equally good. The client is relying on the agent's judgment.
- Keep each property description tight. Clients skim. The showing is where they decide.

## Do Not Include
- Full property report — summary only
- Properties outside the buyer's criteria without explicitly flagging why you included them
- Urgency language unless there is a genuine reason
